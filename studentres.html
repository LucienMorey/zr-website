Rosetta Sheet Named after the famous Rosetta Stone, the Rosetta Sheet translates python and java code into Zero Robotics C.

MIT Tutorials On the MIT website, there are tutorials designed to familiarise a student with some background in coding with Zero Robotics. It covers basic programming, using the IDE, functions, vectors, advanced vector mathematics and code size limitations.

Communication

Communication between team members outside school hours/weekly meetings is especially important during the alliance phase of the International Competition.

We recommend that you use one of these collaborative websites:

Slack Slack in a live chat platform (and more) designed for small engineering teams.
It has integrations to google drive, dropbox and github which allows you to easily share resources and code.
Here is a slack bot by one of our mentors, Luke, which allows you to easily look up game rules in the manual.

Discord Free voice and text chat, designed form gamers.

Google Classroom I have no experience with this, but teams have used it successfully.

Edmodo Like facebook for schools with no direct messaging. Not ideal for Zero Robotics, but it gets through most school firewalls.
